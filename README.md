# Build GCP

Build GCP is a setup wizard which configures GCP Virtual Machine Instance.

## Installation

To use Setup clone the repository to your **/home** directory and;

```bash
$ bash main.sh
```

## Components
- Tesseract-OCR
- Jupyter Notebook
- Tensorflow
- Pytorch
- Docker
- pip
- wget
- setuptools
- leptonica
