# Build GCP

Build GCP is a setup wizard which configures Google Cloud Project Instance.

## Installation

To use Setup clone the main folder to your **/home/** directory and;

```bash
$ bash su.sh
```

## Componenets
- Tesseract-OCR
- Jupyter Notebook
- Tensorflow
- Pytorch
- Docker
- pip
- wget
- setuptools
